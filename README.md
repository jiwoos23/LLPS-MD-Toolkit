# LLPS-MD-Toolkit: Phase Separation & IDR Dynamics Pipeline
An automated, general-purpose computational pipeline for analysing the molecular dynamics, conformational plasticity, and mesoscale phase behaviour of intrinsically disordered proteins (IDPs).

## Overview
This repository contains a robust, automated computational pipeline designed to bridge single-chain atomic structural dynamics with mesoscale liquid-liquid phase separation (LLPS) behaviour. 

While originally developed and validated using the C-terminal intrinsically disordered region (IDR) of Synapsin-1 and the DNAJB6 condensing system, **this pipeline is highly generalisable and can be applied to molecular dynamics (MD) trajectories of any phase-separating protein or IDP.**

## Features
The toolkit integrates monomeric structural analysis with multi-chain macroscopic kinetic tracking:
* **Conformational Plasticity Tracking:** Automated quantification of alpha-helicity (via DSSP), radius of gyration (Rg), end-to-end distance (Ree), Root Mean Square Fluctuation (RMSF), and intramolecular contact maps to detect structural collapses (e.g., liquid-to-solid transitions or rigidification).
* **Macroscopic Condensate Detection:** A spatial clustering script deploying the DBSCAN algorithm. It includes an automated Kneedle-optimised neighbourhood search radius (epsilon) calibration and sensitivity analysis to objectively isolate true macroscopic condensates.
* **Kinetic & Material State Tracking:** Modules to calculate Mean Square Displacement (MSD) for diffusion rates and Bond Survival Probability (S(tau)) to quantify 'sticker' interaction lifetimes and distinguish fluid condensates from kinetically arrested micro-aggregates.
* **Simulation Agnostic:** Designed to process standard trajectory formats (e.g., `.xtc`, `.dcd`) and topology files (e.g., `.pdb`, `.gro`) generated by HyRes-GPU, GROMACS, or OpenMM.

## Prerequisites
To run these scripts, ensure the following dependencies are installed:
* Python 3.8+
* `MDTraj` or `MDAnalysis` (for trajectory handling)
* `NumPy`, `Pandas`, `Matplotlib`, `Seaborn` (for data processing and visualisation)
* `Scikit-learn` (for DBSCAN clustering)
* `kneed` (for automated Kneedle algorithm calibration)

## Repository Structure
* `/analysis_scripts`: Python scripts for DSSP, Rg/Ree, and RMSF calculations.
* `/clustering`: The DBSCAN spatial clustering and boundary analysis scripts.
* `/kinetics`: Scripts for MSD and S(tau) calculations.
* `/example_data`: Sample configurations and topology files demonstrating the required input formats.

## Usage
**1. Monomeric Structural Analysis:**
```bash
python analysis_scripts/calculate_dssp.py --trajectory input.xtc --topology input.pdb --output dssp_results.csv

```
